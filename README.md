# 🧠 Second Brain – Chatbot Bookmark Extension

Second Brain is a Chrome extension that combines a chatbot with a bookmark manager. It allows users to save website bookmarks and later retrieve them by describing the content in natural language.

Instead of remembering URLs, you can simply ask:
“Show me the website about React documentation I saved yesterday”

## Features

- Save web pages with a single click
- Extract and store page content automatically
- Fuzzy search powered by semantic retrieval
- AI-assisted chatbot for knowledge discovery
- Automatic tag generation
- Clickable tag-based filtering
- Fast local-first architecture
- Open saved pages instantly
- Modern dark-themed user interface

## Technology Stack

- React
- TypeScript
- Vite
- Chrome Extension Manifest V3
- Fuse.js
- Chrome Tabs & Scripting APIs
- Local Storage / IndexedDB
- AI-powered retrieval pipeline

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/second-brain.git
cd second-brain
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Extension

```bash
npm run build
```

---

## Load Extension in Chrome

1. Open:

```
chrome://extensions
```

2. Enable:

```
Developer Mode
```

3. Click:

```
Load unpacked
```

4. Select:

```
dist/
```


## Vision

Second Brain is designed to become a personal knowledge retrieval system that helps users organize, search, and converse with everything they save online.
